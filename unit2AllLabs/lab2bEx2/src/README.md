# PetShop

A simple program that allows users to request pets from a pet shop based on their preferences.

The system asks the user for information such as:
- Pet type (cat, dog, bird)
- Pet gender (male, female)
- Pet age (1–24 months)
- Pet color (black, white)

The program then processes the input and provides a pet matching the user's request.
